🐾 Viva Pet Landing Page

Uma landing page moderna e responsiva para um petshop fictício chamado Viva Pet.
O projeto foi desenvolvido utilizando HTML5, CSS3 e JavaScript, com foco em design moderno, organização de componentes e experiência visual acolhedora.

✨ Preview

A landing page possui:

Hero Section moderna
Efeito gradient
Cards de serviços
Navegação com scroll suave
Design responsivo
Estrutura modular com páginas carregadas dinamicamente

🚀 Tecnologias Utilizadas

HTML5
CSS3
JavaScript ES6+
Google Fonts
Font Awesome
